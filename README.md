# QA Specs

Bem-vindo ao repositório do projeto QA Specs! 🚀

## Descrição
Aqui estão os arquivos relacionados ao teste analítico, incluindo scripts, planilhas e criativos.

## Como acessar
Todos os arquivos estão organizados por pastas. Para baixar, clique em **Code → Download ZIP**.

## Estrutura do projeto
- **app.py** → arquivo principal
- **pasta_x** → arquivos auxiliares
- **docs** → documentos de referência

## Observações
- Este repositório é público, então qualquer pessoa pode acessar.  
- Para contribuir, envie um Pull Request ou contate o dono do repositório.
